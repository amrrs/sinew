Chapter 6 Addins

There are two addins in sinew both work off of the same principle. The user highlights a text in the Source editor and sinew will interpret that as the input object of makeOxygen or moga.

To access the addins load sinew (library(sinew)) and then two addins are appended to the Addins menu of RStudio.
